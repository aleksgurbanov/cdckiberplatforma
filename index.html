Fəaliyyət göstərdiyimiz sahələr

Cyber Security

Kibertəhlükəsizlik - internetin istifadəsinin artması bu sahənin də inkişafına öz təsirini göstərdi.Məlumatların saxlanılması nə qədər vacibdirsə,qorunması da bir o qədər vacibdir.Bütün dünyada olduğu kimi Azərbaycanda da bu sahənin inkişaf etməsi üçün,özək hissəmiz olan Kiber Təhlükəsizlik Platforması daxilində fəaliyyətlər yürüdərək bu sahəyə öz töhfəmizi veririk.

Front-End Development

Front-end veb brouzerlərdə görünən hissədir. Həmçinin “client-side” (istifadəçi hissə) də adlandırılır. O özündə yazıları,rəngləri,düymələri,şəkilləri və nəzarətçi meyunu da birləşdirir.Komandamızda digər sahələr kimi bu sahədə də ixtisaslaşmış,peşəkar developerlar fəaliyyər göstərir.

Java Development

Java-nı quraşdırmasanız işlədə bilməyəcəyiniz bir çox proqram və vebsaytlar var və onların sayı günü-gündən artır. Java sürətli, təhlükəsiz və etibarlıdır. Noutbuklardan məlumat saxlama mərkəzlərinə, oyun konsollarından elmi superkompüterlərə, mobil telefonlardan İnternetə qədər Java hər yerdədir! Komandamızda fəaliyyət göstərən Java Developerlər də yeni layihələr yaradaraq Azərbaycanda bu sahənin inkişafına şərait yaradır.

Python Development

Python proqramlaşdırma dili maşın öyrənməsi, sistemlərin avtomatlaşdırılması, veb və API inkişafı və s. üçün əsasdır. Son bir neçə ildə Python müasir proqram təminatının inkişafı, infrastrukturun idarə edilməsi və məlumatların təhlili üçün dünya səviyyəli proqramlaşdırma dili kimi ortaya çıxdı.Komandamız da öz növbəsində Python development sahəsində yeniliklər yaratmaq üçün çalışır.
